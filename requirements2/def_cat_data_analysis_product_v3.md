# Streamlit 簡易アプリの要件定義書
## ゴール: 猫の1日のデータを分析してグラフにするプロダクトをつくりたい
v3 に関するStreamlitの簡易アプリをローカルに作る。上記のゴールを満たす超最低限のStreamlitアプリケーションの要件を記述する。

## 1. 目的
- 猫の1日のデータを分析し、可視化する
- Streamlitを使って簡易的なアプリを作る

## 2. 環境構築方法
- Python 3.9をインストールする
- Streamlitをインストールする
- 仮想環境を作成し、必要なライブラリをインストールする

## 3. 実行方法
- app.pyファイルをStreamlitで実行する
- 必要なデータファイルがある場合は同じディレクトリに配置する

    └── app.py（これは詳細に記載）
