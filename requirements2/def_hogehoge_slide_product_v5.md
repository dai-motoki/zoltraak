## 1. 目的
ユーザーが簡単にスライド資料を作成できるようにするためのStreamlitアプリケーションを開発する。

## 2. 環境構築方法
```
# 仮想環境の作成
python -m venv streamlit_mini
# 仮想環境の有効化
source streamlit_mini/bin/activate
# 必要なライブラリのインストール
pip install streamlit
```

## 3. 実行方法
```
# app.pyファイルの実行
streamlit run app/app.py
```
