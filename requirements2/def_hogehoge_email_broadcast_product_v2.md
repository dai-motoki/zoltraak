## 1. 目的
本アプリケーションの目的は、塾生メール一斉送信プロダクトv2 に関する簡易的な Streamlit アプリを作成することです。

## 2. 環境構築方法
仮想環境 `streamlit_mini` を作成し、以下のライブラリをインストールします。

```
pip install streamlit
```

## 3. 実行方法
作成した `app.py` ファイルを以下のコマンドで実行します。

```
streamlit run app.py
```
